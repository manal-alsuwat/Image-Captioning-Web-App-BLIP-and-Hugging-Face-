Image Captioning Web App (BLIP + Hugging Face)

This project is a simple web application built with Gradio and the BLIP (Bootstrapping Language-Image Pretraining) model from Hugging Face.
It automatically generates descriptive captions for uploaded images, demonstrating how AI can connect visual understanding with natural language generation.
